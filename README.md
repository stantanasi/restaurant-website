## About the project

<p align="center">
  <img src="./.github/docs/preview.png" alt="Preview">
</p>

More than just a portfolio, this project is a **technical showcase** simulating a high-end restaurant platform. It focuses on User Experience (UX), performance, and a cohesive design system.

### Built with

- [Next.js](https://nextjs.org)
- [Material UI](https://mui.com)
- [TypeScript](https://www.typescriptlang.org)

## Getting started

### Prerequisites

Install [Node.js](https://nodejs.org) which includes [Node Package Manager](https://www.npmjs.com/get-npm)

### Setup

1. Clone the project to your local machine

```bash
git clone https://github.com/stantanasi/restaurant-website.git
cd restaurant-website
```

2. Install the packages required

```bash
npm install
```

## Development

Run development mode

```bash
npm run start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser. The page will reload if you make edits.
